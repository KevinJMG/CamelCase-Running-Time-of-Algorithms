## Descripción de los Ejercicios Seleccionados

###  Ejercicio 1: CamelCase (Categoría: Strings)
Objetivo del problema: El ejercicio consiste en analizar una cadena de texto estructurada en el formato de nomenclatura `camelCase` y determinar con precisión cuántas palabras individuales la componen. 

---

### Ejercicio 2: Running Time of Algorithms (Categoría: Sorting)
Objetivo del problema: El propósito de este ejercicio es implementar el algoritmo clásico de ordenamiento por inserción (*Insertion Sort*) y modificarlo para realizar un seguimiento métrico de su rendimiento, contando el número total de desplazamientos (*shifts*) de elementos necesarios para ordenar un arreglo de enteros de forma ascendente.

3. Espera a que cargue el entorno virtual de VS Code en tu navegador.
